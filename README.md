# 坦克大战 - Battle City 🎮

一个 1990 年代风格经典的坦克大战像素游戏，使用纯 HTML/CSS/JavaScript 实现。

## 游戏说明

- **方向键** ← ↑ → ↓ 移动坦克
- **空格键** 射击
- 保护你的基地（鹰旗），消灭所有敌方坦克

## 特性

- 🎯 经典像素风格，26×26 网格地图
- 💥 砖墙和钢墙可破坏
- 👾 三种敌方坦克类型（普通、快速、重甲）
- 🌟 玩家出生无敌保护
- 📈 多关卡连续挑战，关卡越高敌人越多
- 💣 粒子爆炸特效

## 如何游玩

直接用浏览器打开 `index.html` 即可游玩，无需任何依赖。
