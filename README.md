# 坦克大战 - Battle City 🎮

一个 1990 年代风格经典的坦克大战像素游戏，使用纯 HTML/CSS/JavaScript 实现。

## 游戏说明

- **方向键 / WASD** 移动坦克
- **空格键 / J** 射击
- **手机端**：屏幕方向键 + FIRE 圆钮，触屏也能玩
- 保护你的基地（鹰旗），消灭所有敌方坦克

## 特性（当前版本：GBR504/tank-battle 精品覆盖版）

- 🎯 经典像素风格，多关卡地图（4 个关卡可选）
- 💥 砖墙 / 钢墙 + 多种地形障碍
- 👾 4 种敌方坦克类型，AI 各异
- 🌟 玩家出生无敌保护 + 防护盾道具
- 🔧 6 种道具（星 / 炸弹 / 时钟 / 铲子 / 头盔 / 加命等）
- 📱 桌面键盘 + 手机触屏双支持
- 🔊 Web Audio 合成音效（射击 / 爆炸 / 过关等）
- 🏆 最高分本地记录（localStorage）

## 如何游玩

**直接用浏览器双击打开 `index.html` 即可游玩，无需安装任何依赖、无需 npm。**

> ⚠️ 本项目是纯单文件游戏，已移除 esbuild 构建链。请不要执行 `npm install` / `npm run serve`——那套构建残留会导致白屏，直接双击 `index.html` 最稳。
